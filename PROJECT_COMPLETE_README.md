## 🎨 Design System

### Color Palette
```
Primary: #6C63FF (Purple)
Success: #22C55E (Green)
Info: #3B82F6 (Blue)
Warning: #F59E0B (Amber)
Emerald: #10B981
Foreground: #1A1A2E
Background: #FFFFFF
```

### Typography
- **Font Family**: Geist (sans), Geist Mono
- **Headings**: 2xl - 4xl, bold
- **Body**: Regular, muted-foreground secondary text

### Spacing System
- **Border Radius**: 2xl (cards), xl (components), lg (secondary)
- **Padding**: p-4 to p-8
- **Gaps**: gap-3, gap-4, gap-6

### Interactive Elements
- Smooth hover transitions (transition-all)
- Border color changes to primary/50
- Shadow enhancements (hover:shadow-lg)
- Copy feedback ("Copied!" messages)
